文档还很简陋，如果你感兴趣可以帮忙维护！
设置bash环境

cd <starman目录>
./setup.sh
setup.sh脚本将检查系统中的Ruby是否存在，以及版本是否满足要求（>= 2.0.0），然后在~/.bashrc添加设置语句。

配置STARMAN

starman config
主要修改package_root和install_root（注：如果设在/opt/starman/packages和/opt/starman/software，可以直接使用已经预编译的软件包，省去自己编译），以及设置编译器集合。

安装软件包

starman install

生成软件包shell配置文件

starman shell -update-config [package options]

进入软件包环境

starman shell
